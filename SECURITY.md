# Security Policy

## Reporting a Vulnerability

We take the security of this project seriously. Please do not report security vulnerabilities through public GitHub issues.

Instead, please report them privately using one of the following methods:

1. **GitHub Private Vulnerability Reporting**: This is the preferred method. Use GitHub's built-in private advisory feature in the repository's "Security" tab.
2. **Email**: Alternatively, you can contact **rutova2@gmail.com** directly.

We will acknowledge receipt of your vulnerability report within **48 hours**.
